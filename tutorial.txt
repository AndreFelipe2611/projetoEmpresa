1- instalar o xampp:
digitar xampp download no google, instalar a versão 8.2 que é a mais recente pra não ter problemas
(aceite tudo que o xampp for oferecendo)

2- instalar mySQL (banco de dados)
video do tutorial: https://www.youtube.com/watch?v=IEUgVwjXF0o&t=541s

3- quando tiver os dois, entra xampp no DISCO LOCAl, entra na pasta htdocs e joga o projeto dentro dela

4- no xampp ligue o apache e o mySQL  (caso o mySQL não ligue, relaxa, que o banco funciona offiline)

5-no google digita localhost/o nome da pasta para abrir

6- caso queira rodar o banco no vscode (maior trampo), siga o tutorial de 2 minutos: https://www.youtube.com/watch?v=cE8V3EJ0h7E

7-criei uma basta banco de dados pra vc jogar no mySQL depois
